## 模型一览
### 第一层：传统基线
─────────────────────────

Popularity   → 全平台热销榜，所有人都推一样的，纯地板

UserCF       → "和你行为相似的用户买了什么"，用户间协同过滤

ItemCF       → "你买的这个东西，别人还一起买了什么"，商品间协同过滤

### 第二层：深度学习
NCF          → 用 MLP 学 user-item 交互，替代传统矩阵分解，深度基线

第三层：创新模型

NCF+Review   → NCF + 评论文本语义融合，验证"语文有没有帮助"

LightGCN     → 在 user-item 二部图上传播 embedding，验证"图结构有没有帮助"

### 每个模型回答什么问题
对比	回答

Popularity vs 其他所有	你的系统比"推荐畅销品"好多少

UserCF / ItemCF vs NCF	传统方法到头了，深度学习有必要吗

NCF vs NCF+Review	RQ2 — 评论语义能不能提升推荐

NCF vs LightGCN	RQ3 — 图结构能不能更好建模用户-商品关系
